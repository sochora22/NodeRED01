
# Zadání

1. Spusťte NodeRED

2. Naimportujte flow ze souboru flow.json

3. Zjistěte aktuální:

   * teplotu
   * tlak

z měřáků, uložte screenshot měřáků do souboru gauges.png a proveďte váš první git commit ve staženém repozitáři.

4. Přidejte debug nody tak, aby se vpanelu debug zobrazoval:

   * complete msg object
   * msg.payload.D (směr větru)
   * msg.payload.F (síla větru)

uložte screenshot flow s debugovacími údaji v pravém panelu a proveďte váš druhý git commit.

5. Upravte flow tak, aby zobrazovalo také sílu větru (F) na třetím gauge. Exportujte toto flow do souboru my_flow.json a proveďte váš třetí git commit.

6. Proveďte push repozitáře na váš GitHub a odkaz na vámi vytvořený repozitář odevzdejte do odevzdávacího adresáře na společném disku.
